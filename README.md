# Enterprise-Resource-Planning-in-Python-Database


## 🎠 About

- 데이터의 실시간 업데이트와 협업에 포커싱한 의류매장 ERP
- 기존의 여러 프로그램을 사용하는 의류 매장은 통합되지 않는 시스템으로 인한 불편함이 존재하여 데이터의 실시간 업데이트와 업무최소화를 고안한 의류매장 ERP 시스템입니다.
